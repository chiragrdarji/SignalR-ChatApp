GANERAL USAGE INSTRUCTIONS
-------------------------------------------------
1. Type your name to log in.
2. To test chat application, open multiple browsers and type different name in each one. Make sure you open different browser not tab in same browser.
3. As this is demo code, name uniqueness validation is not implemented.
4. Allow up to 20 users.
5. Double click on user name to initiate private chat.
6. When user joins chat room he/she will see last 20 conversion of group chat.
7. Display user name and message when other user is typing.
8. When user left chat room, it shows notification about who left room.
